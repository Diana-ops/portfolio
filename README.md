## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/Diana-ops/portfolio
```
Acesse a pasta
```
cd portfolio
```
Instale as dependencias 
```
yarn install
```
Execute a aplicação
```
yarn start
```
